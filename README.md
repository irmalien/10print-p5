# 10Print
10print is based on historical single line Commodore 64 BASIC code with same name. 10print program generates multiple Truchet tiles with various scale of detailing, color and size creating limitless and unique graphical patterns. User can change, color scheme, randomness mode as well pause and download resulting image.

[Visit site](http://www.mvcd.se/projects/10print/index.html)

2. Fiddle around with settings.
3. Download result as png image file.
